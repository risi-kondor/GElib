Clebsch–Gordan matrices¶

Given two irreps and of SO(3), the tensor product representation reduces into a direct sum of irreps in the form

called the Clebsch–Gordan decomposition. The submatrix of corresponding to a particular value of we will denote .

Taking the Clebsch–Gordan product of two (or more) SO3vecs is an equivariant operation and is an important building block of group equivariant neural networks.

The function SO3ClebschGordanTensor(l1,l2,l) returns as a third order tensor.

>>> C=gelib.SO3CGtensor(1,2,2)
>>> C
tensor([[[ 0.0000,  0.0000,  0.0000,  0.0000,  0.0000],
         [-0.5774,  0.0000,  0.0000,  0.0000,  0.0000],
         [ 0.0000, -0.7071,  0.0000,  0.0000,  0.0000],
         [ 0.0000,  0.0000, -0.7071,  0.0000,  0.0000],
         [ 0.0000,  0.0000,  0.0000, -0.5774,  0.0000]],

        [[ 0.8165,  0.0000,  0.0000,  0.0000,  0.0000],
         [ 0.0000,  0.4082,  0.0000,  0.0000,  0.0000],
         [ 0.0000,  0.0000,  0.0000,  0.0000,  0.0000],
         [ 0.0000,  0.0000,  0.0000, -0.4082,  0.0000],
         [ 0.0000,  0.0000,  0.0000,  0.0000, -0.8165]],

        [[ 0.0000,  0.5774,  0.0000,  0.0000,  0.0000],
         [ 0.0000,  0.0000,  0.7071,  0.0000,  0.0000],
         [ 0.0000,  0.0000,  0.0000,  0.7071,  0.0000],
         [ 0.0000,  0.0000,  0.0000,  0.0000,  0.5774],
         [ 0.0000,  0.0000,  0.0000,  0.0000,  0.0000]]])
