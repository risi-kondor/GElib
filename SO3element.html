SO3elementÂ¶

An SO3element object represents a single element of the rotation group . The group element can either be initialized from the Euler angles or chosen randomly from the group.

>>> a=gelib.SO3element(0.2,1.0,-0.1)
>>> a
SO3element(0.200000,1.000000,-0.100000)
>>> b=gelib.SO3element.uniform()
>>> b
SO3element(0.534275,2.801079,1.191856)

The rho method returns the representation matrix (Wigner D-matrix) . Since is a dimensional representation, the result is a dimensional complex matrix.

>>> a=b.rho(2)
>>> print(a)
