An SO3type object specifies the multiplicites of each irreducible representation in an SO3-vector.

>>> tau=gelib.SO3type([3,4,1]) # Define tau=(3,4,1)
>>> tau
<GElib::SO3type(3,4,1)>
>>> print(tau)
(3,4,1)
>>> tau[2]               # Get tau_2
1
>>> tau[2]=3             # Set tau_2=3
>>> tau
<GElib::SO3type(3,4,3)>
>>> tau.maxl()           # Print out the index of the largest irrep
2

The function CGproduct computes the type of the ClebschâGordan product of two SO-vectors of given types.

>>> tau=gelib.SO3type([2,2])
>>> mu=gelib.CGproduct(tau,tau)
>>> print(mu)
(8,12,4)

The optional argument maxl limits the product to a given maximal .

>>> tau=gelib.SO3type([1,1,1])
>>> print(gelib.CGproduct(tau,tau))
(3,6,6,3,1)
>>> print(gelib.CGproduct(tau,tau,2))
(3,6,6)
