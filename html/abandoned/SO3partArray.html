SO3partArr¶

Similary to the tensor array classes in cnine, the basic tool for parallelizing SO3-operations in GElib are the SO3partArr and SO3vecArr classes. An SO3partArr is a multidimensional array of SO3part objects with the same and .

GPU operations¶

SO3partArr objects can be moved to/from the GPU the same way as SO3part and SO3vec objects.

The benefits of GPU acceleration for arrayed objects is particularly great, because the corresponding kernels (specifically, kernels implementing Clebsch-Gordan operations) are written in a way as to parallelize the operation across array cells.
